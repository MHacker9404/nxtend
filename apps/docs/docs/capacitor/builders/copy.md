Runs a builder that copies an applications build output to a Capacitor project native platform.

## Usage

```
nx run {capacitor project}:copy --platform {ios, android}
```

## Properties

### --platform

Default: `null`

Type: `string`

Possible values: `ios`, `android`

The native platform to copy.
